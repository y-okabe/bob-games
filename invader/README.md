# Space Invaders - フル機能版

HTML、CSS、JavaScriptで作成した本格的なインベーダーゲームです。

## 🎮 機能

### 基本機能
- **プレイヤー操作**: 左右移動と射撃
- **敵の編隊**: 5行11列の敵編隊が左右に移動しながら攻撃
- **衝突判定**: プレイヤー、敵、弾丸の正確な衝突判定
- **スコアシステム**: リアルタイムスコア表示とハイスコア記録（LocalStorage使用）

### 拡張機能
- **ライフシステム**: 3-5ライフ（難易度による）、被弾後の無敵時間
- **レベルシステム**: 10レベル制、レベルが上がるごとに難易度上昇
- **難易度選択**: Easy、Normal、Hardの3段階
- **パワーアップシステム**:
  - 🔴 **連射 (R)**: 射撃間隔が短縮（10秒間）
  - 🔵 **シールド (S)**: 無敵状態（8秒間）
  - 🟡 **スコアブースト (B)**: スコア2倍（15秒間）
  - 🟣 **ワイドショット (W)**: 3方向同時射撃（12秒間）

### UI/UX機能
- **テーマシステム**: 3種類のビジュアルテーマ
  - レトロ: クラシックな緑色のドット絵風
  - モダン: 青を基調とした現代的なデザイン
  - ネオン: ピンクのネオン風デザイン
- **サウンドエフェクト**: Web Audio APIによる効果音
  - 射撃音、爆発音、パワーアップ取得音
  - レベルクリア、ゲームオーバー、ゲームクリア音
- **パーティクルエフェクト**: 爆発時のパーティクルアニメーション
- **レスポンシブデザイン**: モバイルデバイスにも対応

## 🎯 操作方法

### キーボード操作
- **← →**: プレイヤーの左右移動
- **SPACE**: 射撃
- **ESC / P**: 一時停止
- **Enter**: 射撃（代替キー）

### ゲームの流れ
1. タイトル画面で「ゲームスタート」をクリック
2. 難易度とテーマを選択（オプション）
3. 敵を全滅させてレベルクリア
4. レベル10をクリアするとゲームクリア

## 🎨 テーマ

### レトロテーマ（デフォルト）
- クラシックなアーケードゲーム風
- 緑色のモノクロームデザイン
- ドット絵風のグラフィック

### モダンテーマ
- 現代的な青を基調としたデザイン
- グラデーションと影を使用
- 滑らかなアニメーション

### ネオンテーマ
- サイバーパンク風のピンクネオン
- 強い発光エフェクト
- 暗い背景に映えるデザイン

## 🎯 難易度

### Easy（初心者向け）
- 敵の移動速度: 遅い
- 敵の攻撃頻度: 低い
- プレイヤーライフ: 5

### Normal（標準）
- 敵の移動速度: 標準
- 敵の攻撃頻度: 標準
- プレイヤーライフ: 3

### Hard（上級者向け）
- 敵の移動速度: 速い
- 敵の攻撃頻度: 高い
- プレイヤーライフ: 3

## 🏆 スコアシステム

### 敵の撃破ポイント
- 最上段（タコ型）: 30点
- 2-3段目（カニ型）: 20点
- 4-5段目（イカ型）: 10点

### ボーナス
- レベルクリアボーナス: 1,000点
- スコアブーストパワーアップ中: 全ポイント2倍

### ハイスコア
- ブラウザのLocalStorageに保存
- ゲーム終了時に自動更新

## 🔧 カスタマイズ

`js/config.js`ファイルを編集することで、以下の設定を変更できます：

- キャンバスサイズ
- プレイヤーの速度と射撃間隔
- 敵の数、速度、攻撃頻度
- パワーアップの効果時間と効果
- レベル数と難易度曲線
- 色設定

## 🐛 デバッグモード

`js/config.js`の`DEBUG`セクションで以下を有効化できます：

```javascript
DEBUG: {
    SHOW_FPS: true,        // FPS表示
    SHOW_HITBOXES: true,   // ヒットボックス表示
    GOD_MODE: true         // 無敵モード
}
```
